# きょうりゅうカタカナトレーニング

5歳の子どもが恐竜をテーマにカタカナを楽しく練習できるスマホ向けWebアプリです。

## 対象ユーザー

- 5歳の男の子（カタカナ練習中）
- 主な利用端末：Pixel 7a（Chrome）
- 恐竜が大好きなお子さん

## 機能一覧

| 機能 | 説明 |
|------|------|
| タイトル画面 | 恐竜・ジャングル・火山テーマのトップ |
| ステージ選択 | ア行〜ワ行の10ステージ（初期3ステージ解放） |
| 難易度選択 | たまごモード / こども恐竜モード / つよい恐竜モード |
| 3択クイズ | 1ステージ5問。正解・不正解アニメーション付き |
| 結果画面 | スコア表示、4問以上でステージクリア |
| 恐竜カード | ステージクリアで恐竜カードを1枚獲得 |
| 恐竜図鑑 | 獲得済みカードを一覧。未獲得は？？？表示 |
| 復習モード | 間違えたカタカナを優先出題 |
| 効果音 | Web Audio APIによるシンプルなビープ音 |
| データ保存 | localStorage（クリア済みステージ、カード、間違いメモ） |

### 難易度

- **たまごモード**：表示されたカタカナと同じ文字を3択から選ぶ
- **こども恐竜モード**：ひらがなの読み方に対応するカタカナを選ぶ
- **つよい恐竜モード**：恐竜・日常の言葉からカタカナを選ぶ

### 恐竜カード

4問以上正解でステージクリア扱いとなり、1枚獲得できます。

| カード | 対応ステージ |
|--------|--------------|
| ティラノサウルス | アぎょう |
| トリケラトプス | カぎょう |
| プテラノドン | サぎょう |
| ステゴサウルス | タぎょう |
| ブラキオサウルス | ナぎょう |
| ヴェロキラプトル | ハぎょう |

## ファイル構成

```
├── index.html   # HTMLマークアップ（全画面）
├── style.css    # スタイル・アニメーション

```

## ローカルでの開き方

```bash
# index.html をブラウザで直接開くだけでOK
open dino-katakana-app/index.html
```

または VS Code の Live Server 拡張機能を使うと便利です。

> **Note:** ファイルを直接開く（`file://`）だけで動作します。サーバー不要です。

## GitHub Pages での公開手順

1. このリポジトリを GitHub にプッシュする
2. GitHub リポジトリの **Settings** → **Pages** を開く
3. **Source** を `Deploy from a branch` に設定
4. **Branch** を `main`（または任意のブランチ）、フォルダを `/ (root)` に設定
5. **Save** をクリック

公開後のURL例：
```
https://<ユーザー名>.github.io/<リポジトリ名>/dino-katakana-app/
```

> すべてのパスが相対パスのため、サブディレクトリ公開でも問題なく動作します。

## localStorage の保存内容

| キー | 内容 |
|------|------|
| `dinoKatakana_v1` | JSON形式で以下を一括保存 |
| `unlockedStages` | 解放済みステージID配列 |
| `clearedStages` | クリア済みステージID配列 |
| `ownedCards` | 獲得済みカードID配列 |
| `wrongChars` | 間違えたカタカナ配列 |
| `soundOn` | 効果音ON/OFFフラグ |
